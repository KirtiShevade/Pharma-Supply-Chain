# Pharma Inventory Risk Dashboard

## Problem Statement
Pharma procurement teams have no real-time visibility into which drugs are running critically low they react after the shortage hits, not before.

## Description
This dashboard flags drugs at critical and low stock 
levels across warehouses and suppliers giving procurement managers a single view to act before stockouts occur.

## Tech Stack
- Python → synthetic data generation
- SQL Server → data storage and retrieval
- Power Query → data cleaning and transformation
- Power BI → visualization and dashboard building

## Data Source
Synthetically generated using Anthropic Claude AI. 320 records across 40 drugs, 8 warehouses, 10 suppliers.

## Features & Highlights
- Flags CRITICAL and LOW stock drugs instantly
- Stacked bar chart showing risk by supplier
- Stacked bar chart showing risk by warehouse city
- Drill-down table with drug, dosage, days of stock remaining and lead time
- Interactive slicers for warehouse city and supplier

## Disclaimer
Data is synthetically generated for learning purposes only. Does not represent actual inventory data of any company.
